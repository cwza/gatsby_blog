## Install from via homebrew
* `brew install llvm`
* clang++ and clangd binary will be installed in `/usr/local/opt/llvm/bin`
* std headers will be installed in `/usr/local/opt/llvm/include/c++/v1`
* `clang++ -E -x c++ - -v < /dev/null` to see the clang++ infos
* Add to your .zshrc: `export PATH="/usr/local/opt/llvm/bin:$PATH"`

## (Optional) Install fmt
* `brew install fmt`
``` cpp
#include<fmt/core.h>
#include<fmt/ranges.h>
vector<int> a = {1, 2, 3};
fmt::print("{}\n", a); // {1, 2, 3}
```

## How to compile
``` bash
# -g: Generate debug information
# -O2: Optimization level 2
# -Wall: Display all warnings;
# -fsanitize=addressa: Check array, vector index out of range at run time
# -isystem/usr/local/include: Add include path
# -L/usr/local/lib: Add link path
# If you have installed fmt, use -lfmt to link it
clang++ "$(fname)" -o a.out -std=c++17 -g -Wall -fsanitize=address -isystem/usr/local/include -L/usr/local/lib -lfmt
```
